PAGÍNAS WEB: CONSTRUCCIÓN

Estáticas


Los sitios web estáticos están formados por archivos almacenados en el servidor que se muestran tal cual en el navegador del usuario. Están creados con lenguajes del lado del cliente como HTML y CSS, y su contenido no cambia automáticamente.

Las formas más comunes de crear una web estática son:

  • Codificar desde cero, usando HTML, CSS y JavaScript, recomendado para desarrolladores con experiencia.

  • Usar generadores de sitios estáticos, que crean páginas HTML a partir de plantillas, facilitando y acelerando el proceso.

  • Plataformas de creación de sitios web, que ofrecen editores visuales y no requieren conocimientos de programación, ideales para principiantes.

 

 Dinámicas


Las páginas web dinámicas utilizan lenguajes de programación que separan la estructura fija del contenido, el cual se carga en el momento en que el usuario accede a la página. La información suele almacenarse en bases de datos y gestionarse mediante gestores de contenido (CMS).

Los lenguajes más utilizados son PHP, ASP y JSP:

  • PHP es un lenguaje libre y multiplataforma que permite trabajar con distintas bases de datos. Funciona del lado del servidor y no puede ejecutarse directamente en el navegador.

  • ASP es una tecnología de Microsoft para crear páginas dinámicas, similar a Visual Basic, que también permite el acceso a bases de datos.

  • JSP es una tecnología basada en Java que requiere la máquina virtual de Java y permite crear aplicaciones web dinámicas mediante scripts y librerías, incluyendo pequeños programas llamados applets.
